Estrutura do HTML

(Voltar)

HTML (abreviação de Hypertext Markup Language) é uma linguagem de marcação utilizada na estruturação de páginas web, atualmente usa-se a versão do HTML 5 . Para iniciar um projeto no html, é necessário criar uma pasta no seu computador, direcionada para seus arquivos, imagens e tudo que for preciso para sua criação. Depois é abrir o Visual Studio Code VS Code, lá abre-se a pasta criada, nela criaremos o nosso arquivo index.html,geralmente a página principal de um site é chamada de index. A página html consiste em três pontos principais, aliás três tags (html, head e body) (html - estrutura da página) (head - cabeçalho) (body - corpo da página), todas elas terão os sinais de maior e menor (< e >) com abertura e fechamento (< e />).

TAGS

(Voltar)

Todas páginas html são compostas por tags,tudo que tiver entre elas, é informado ao navegador o que está sendo construído, para que ele interprete e execute corretamente o conteúdo, em algumas delas podemos usar atributos, definidos pelo símbolo de igual (=) e separados por espaços. Os atributos podem ter valores atribuídos a eles, que devem estar entre aspas duplas (“).

Textos

(Voltar)

Para colocar texto no corpo do site, usamos algumas tags, elas sinalizam para o navegador a importância de cada uma. São diversas as tags que podemos utilizar, para diversas variações no texto,nunca esquecer do sinal de maior e menor, abrir e fechar as tags.

Links

(Voltar)

Os links são inseridos no texto através da tag (< a >), dentro dessa tag colocamos o atributo (href), que é o endereço do link, dessa forma criamos o link que abrirá na mesma página que estamos navegando, se quiser que ele abra em uma nova aba, usa-se o atributo (target).
Exemplo, mesma aba:( < a href="url"> Site < /a > )
Exemplo, outra aba:( < a href="url" target=""> Site < /a > )
